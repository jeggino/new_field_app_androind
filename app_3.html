How to use the app

  • Create new project starts an empty GeoJSON FeatureCollection.
  • Open existing project loads a GeoJSON file from your device.
  • Use the circular + button to start adding an observation.
  • A cross appears in the center of the map. Move the map so the cross is exactly where the observation belongs.
  • Choose whether the observation is a bat or a bird, select the species, date, time, and add comments.
  • Click an observation marker to edit or delete it.
  • The app asks for confirmation before editing or deleting an observation.
  • Whenever you add, edit, or delete, the GeoJSON is updated. If direct file saving is unavailable, an updated copy is downloaded.
