How to use the app

  • Start by choosing Create new project or Open existing project.
  • A project is saved as a GeoJSON FeatureCollection.
  • Use the round + button to add a point observation.
  • Use the GPS button in the bottom-left corner to zoom to your current location.
  • Use the map layer button to switch between street map and satellite map.
  • Use the sidebar drawing tools to add a red research area, observation area, or foraging route.
  • Research areas are red outlines with no fill.
  • Observation areas can be saved as Foraging area or Pair area.
  • Foraging routes include species, date, time, and comments.
  • Different species are automatically shown with different colors.
  • Use the Project Summary button to open charts and project statistics.
  • Click a feature on the map to edit or delete it.
