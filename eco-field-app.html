How to use the app

  • Start by choosing Create new project or Open existing project.
  • A project is a GeoJSON FeatureCollection file.
  • Use the round + button to add a field observation.
  • A cross appears in the center of the map. Move the map until the cross is on the observation location.
  • Select bat or bird, choose the species, date, time, and add comments.
  • Click an observation marker to edit or delete it.
  • The app asks for confirmation before editing or deleting.
  • After every add, edit, or delete, the GeoJSON file is updated or an updated copy is downloaded.
