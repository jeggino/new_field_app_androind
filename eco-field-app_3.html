How to use the app

  • Create new project creates a new empty/strong> button in the bottom-right corner to add an observation.
  • Use the live location button in the bottom-left corner to jump to your current position.
  • Use Draw polygon in the sidebar to create an area feature.
  • Click an observation marker to edit or delete it.
  • The app asks for confirmation before editing or deleting an observation.
  • GeoJSON points, lines, polygons, and multi-geometries are drawn on the map.
  • Use the layer control in the top-right of the map to switch between standard and satellite maps.
