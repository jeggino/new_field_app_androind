How to use the app

  • Start by choosing Create new project or Open existing project.
  • A project is saved as a GeoJSON FeatureCollection.
  • Use the round + button to add a point observation.
  • Use the GPS button to zoom to your current location.
  • Use the sidebar drawing tools to add a research area, observation area, or survey line.
  • For point observations, move the map until the crosshair is over the observation location.
  • Click a point, polygon, or line on the map to delete it. Point observations can also be edited.
  • After every add, edit, or delete, the GeoJSON file is updated or an updated copy is downloaded.
